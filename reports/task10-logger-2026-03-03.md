## Implementation Details

### New production files

- `ExcelReport/ExcelReportLib/Logger/IReportLogger.cs`
- `ExcelReport/ExcelReportLib/Logger/ReportLogger.cs`
- `ExcelReport/ExcelReportLib/Logger/LogEntry.cs`
- `ExcelReport/ExcelReportLib/Logger/LogLevel.cs`
- `ExcelReport/ExcelReportLib/Logger/ReportPhase.cs`

### Behavior implemented

- Supports `Debug`, `Info`, `Warning`, and `Error` levels.
- Stores optional `ReportPhase` on each log entry.
- Supports low-level entry logging and convenience helpers.
- Supports `Issue`-backed logging via `LogIssue`.
- Returns snapshots for all entries and level-filtered entries.
- Produces a chronological audit trail by sorting on `Timestamp`.
- Clears all buffered entries with `Clear()`.
- Uses a lock around the backing list for thread-safe reads and writes.
